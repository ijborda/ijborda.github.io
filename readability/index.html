How to interpret?

The result approximates the U.S. grade level thought necessary to comprehend the text. The lower the grade level is, the easier the text is comprehend.

How is it calculated?

The test uses the Coleman–Liau index (CLI) to calculate the grade level. It uses the following formula: \(CLI = 0.0588L - 0.296S - 15.8\). \(L\) is the average number of letters per 100 words and \(S\) is the average number of sentences per 100 words.
