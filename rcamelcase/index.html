How to Use?

  1. Enter column names in the box above. Each line corresponds to one column name.
  2. The number of column names entered so far is shown below the box.
  3. Click "Submit" to convert the column names to R ready vector formatted in camelCase.
  4. Click "Copy Text" to copy the output to your clipboard.

How does it work?

  1. It starts by removing all non-alphanumeric characters.
  2. White spaces before and after each column names are also stripped.
  3. The first character of every word (except the first word) is set to uppercase, all other characters are set to lowercase.
  4. It assumes that words are separated by either space or underscore.
  5. All white spaces are then removed.
  6. Finally, the output is formatted to be R-ready by formatting it to "c("colname1", "colname2", "colname3", ...)".

Example:

Input Text:

First Name
Last NAME
"Nationality"
Age
Date_of_Birth
GEndER

Output Text:

c("firstName", "lastName", "nationality", "age", "dateOfBirth", "gender")
